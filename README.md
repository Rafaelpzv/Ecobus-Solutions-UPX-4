# 🚌 EcoBus Solutions

Sistema web para melhoria da mobilidade urbana, conectando passageiros e motoristas em tempo real através de sinais digitais nos pontos de ônibus.

---

## 📌 Sobre o Projeto

O **EcoBus Solutions** é uma aplicação desenvolvida com foco em tornar o transporte público mais eficiente e sustentável.

A plataforma permite que passageiros sinalizem sua presença nos pontos de ônibus, possibilitando que motoristas visualizem a demanda em tempo real e tomem decisões mais inteligentes durante a rota.

---

## 🎯 Objetivo

* Reduzir perda de viagens
* Evitar ônibus passando direto em pontos com passageiros
* Melhorar a comunicação entre usuários e operação
* Otimizar rotas com base em demanda real

---

## ⚙️ Funcionalidades

* 📍 Emissão de sinal de presença em pontos
* 🗺️ Visualização de rotas e mapa em tempo real
* 🚌 Painel do motorista com alertas
* 📊 Dashboard administrativo
* 📋 Histórico de embarques
* 🔔 Notificações em tempo real

## 🚀 Como rodar o projeto

```bash
# Clonar repositório
git clone https://github.com/Rafaelpzv/Ecobus-Solutions-UPX-4.git

# Entrar na pasta
cd Ecobus-Solutions-UPX-4

# Instalar dependências
npm install

# Rodar projeto
ng serve
```

Acesse em:

```
http://localhost:4200
```
